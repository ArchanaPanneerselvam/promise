Question 4:

"Create a promise function accepting an argument, if yes is passed to the function then it should go to resolved state and print Promise Resolved, and if nothing is passed then it should go to reject the state and catch the error and print Promise Rejected "

                    var ans = function (yes) {
                        return new Promise(function (resolve, reject) {
                            if (yes) {
                                resolve("Promise Resolved")
                            } else {
                                reject(" Promise Rejected")
                            }
                        });
                    };
                    ans(true).then(function (answer) {
                        console.log(answer);
                    }).catch(function (err) {
                        console.warn(err);
                    });

O/P: Promise Resolved

Reason: The function called ans() that creates a new Promise for you when it’s run.You can pass in a boolean (true/false) argument about whether or not to reveal the answer. The Promise will resolve() if it’s true, and reject() if it’s not.We’ll use it to create a new Promise, and add .then() and catch() callback methods to log.
